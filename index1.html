Encapsulation

Encapsulation is basically the procedure in
which we group the related data items and
the functions operation on them together. This
allows us the ability that the private variables
are visible only in the cuurent function and
cannot be accessed by other function or global scope.

For Example:-

const person = function(name,age){
let name = name;
let age = age;
return {
description:function(){
console.log(`My name is ${name} and age is ${age}`);
}
}
}
const myself = new person("sachin",22);
myself.description();
//My name is sachin and age is 22

Abstraction

Abstraction is basically a way of creating simple
models outs of complex models by including only the
important properties which are needed to build the
model.
Therefore the main idea of abstraction is to define
real-lifr components in various data type hich are
available.

For Example:-

function person(name, age, email, height, weight){
this.name = name;
this.age = age;
this.email = email;
this.height = height;
this.weight = weight;
}
const binod = new person("sachin", 22, "sachinchourasiya.c.123@gmail.com",174,64);

Inheritance

Inheritance is basically the way through
which we share common functionalities
among classes. So in other ways, it gives
the power to avoid code duplication in a
class, by borrowing the functionalities from
the other class which prossesses those
functionalities.

For Example:-

Polymorphism

Polymorphism is the technique through which we can
substitute classes that have common functionality in
terms of their properties and methods.

For Example:-

function Animal(sound){
this.sound = sound;
this.speak=function(){
return this.sound;
}
}
function showInfo(obj){
console.log(obj.speak());
}
//different object
var dog =new Animal("woof");
var cat =new Animal("meow");
//responds different ways
showInfo(dog);
showInfo(cat);

Inheritance

Javascript classes supports inheritance
which means we can borrow properties
and methods from other classes.

For Example:-

class Animal {
constructor(name){
this.speed = 0;
this.name =name;
}
run(speed){
this.speed = speed;
console.log(`${this.name} runs with a speed of ${this.speed} `);
}
stop(){
this.speed = 0;
console.log(`${this.name} stands still`);
}
}
class Cat extends Animal{
hide(){
console.log(`${this.name} hides!`)
}
}
let cheetah = new Cat('Cheetah');
cheetah.run(100);//Cheetah runs with a speed of 100
cheetah.hide();//cheetah hides

Super

The Super keyword is used to access and
call function which are present in the
parent's object.

For Example:-

class Car {
constructor(brand){
this.carname = brand;
}
present(){
return "i have a"+this.carname;
}
}
class Model extends Car {
constructor(brand , mod){
super(brand);
this.model=mod;
}
show(){
return this.present()+", it is from"+ this.model;
}
}
mycar = new Model("Maruti 800","2004");
mycar.show()//i have maruti 800, it is from 2004
